## 루브릭 v2 — 10개 차원 (총 50점)

> 각 차원은 체크리스트 항목으로 구성됩니다. 항목마다 ✅/❌로 표기하고, 충족된 항목 수로 점수를 계산합니다.

---

### D1. 핵심 기능 (Core Functionality) — 10점

**체크리스트 (각 1점):**
- [ ] 7탭 전부 렌더링 (대시보드·단가네고·계약·선적·통관·재고·품목이력)
- [ ] 인라인 편집 모드 토글 (editMode 전역 상태)
- [ ] 셀 클릭 → 타입별 편집 (텍스트·숫자·날짜·드롭다운 4가지)
- [ ] 오더 추가 모달 (FAB 버튼 + 6개 섹션 폼)
- [ ] 오더 수정 모달 (기존 데이터 프리필)
- [ ] 오더 삭제 (확인 다이얼로그 포함)
- [ ] 엑셀 내보내기 3시트 (오더·재고·시장가)
- [ ] 이메일 알림 발송 (ETA 임박 + 안전재고 미달 감지)
- [ ] Chart.js 차트 최소 2개 이상 (월별 시장가 추이 등)
- [ ] 긴급 알림 뱃지 (탭 배지 또는 대시보드 알림 카드)

**점수 = 충족 항목 수 × 1점**

- [ ] 상태값 허용 목록 검증 (`negotiation.status`, `customs.status`, `dispatch.status`)
- [ ] 검증 실패 시 저장 차단 (exit 2 또는 프론트 경고)

**체크리스트 (각 1점):**
- [ ] 오더 stage 7단계 순서 반영 (네고중→계약완료→발주대기→선적완료→항해중→통관중→배차완료)

- [ ] 안전재고 대비 현재고 부족률 계산·표시
- [ ] 협상 절감액 계산 (오퍼가 - 계약가) 표시
