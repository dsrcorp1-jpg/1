## 왜 이 프로젝트가 존재하는가

PP/PE 레진 구매는 매달 수억 원의 의사결정이다.  
하지만 가격 정보는 유료 구독 서비스에 분산되어 있고,  
유가-나프타-레진의 연쇄 구조는 직관적으로 이해하기 어려우며,  
공급사마다 오퍼 조건이 달라 비교가 번거롭다.

이 프로젝트는 그 정보의 불균형을 줄이기 위해 시작되었다.

---

## 핵심 가치

### 1. 투명성 (Transparency)
단가 산정 근거를 공식으로 문서화한다.  
`PP 계약가 = 나프타 × 1.55 + $85 + $20`처럼 누구나 검증 가능해야 한다.  
블랙박스 협상이 아닌, 근거 있는 협상을 지향한다.

### 2. 지속성 (Continuity)
매월 수동으로 복사-붙여넣기하는 작업을 없앤다.  
GitHub Actions가 자동으로 데이터를 수집하고, Excel이 자동으로 행을 추가하며,  
GitHub Issues가 자동으로 체크리스트를 생성한다.

### 3. 맥락 (Context)
숫자 하나가 아니라 그 숫자가 어디서 왔는지를 함께 전달한다.  
유가가 오른 이유, 스프레드가 좁아진 배경, 환율 충격의 크기.  
차트는 그 맥락을 한눈에 보여주기 위해 존재한다.

### 4. 실용성 (Practicality)
완벽한 시스템보다 지금 당장 쓸 수 있는 도구가 낫다.  
API 키가 없어도 근사치 공식으로 동작하고,  
Excel이 없어도 브라우저에서 단가를 입력할 수 있도록 설계했다.

## 데이터 철학

### 실거래가는 여기 없다
이 리포트의 모든 가격은 **ICIS·Platts·ChemOrbis 참조 기반 재구성 시계열**이다.  
실제 계약 단가는 별도 협상과 구독 서비스를 통해 확인해야 한다.  
이 도구는 방향을 잡기 위한 나침반이지, GPS가 아니다.

### 상관관계는 인과관계가 아니다
유가와 레진의 상관계수 r = 0.87은 강한 연관성을 뜻하지만,  
유가가 오른다고 레진이 반드시 오르는 것은 아니다.  
중국 수요, 설비 증설, 물류 변수가 단기에는 유가를 압도한다.  
차트는 그 복잡성을 단순화하지 않기 위해 설계되었다.
